Linguagens de programação

Diversas linguagens de programação surgiram desde então. Entre estas incluem-se C#, VB.NET, Java, Object Pascal, Objective-C, PHP, Python, SuperCollider, linguagem D e Ruby.

PHP

O PHP (um acrônimo recursivo para PHP: Hypertext Preprocessor) é uma linguagem de script open source de uso geral, muito utilizada, e especialmente adequada para o desenvolvimento web e que pode ser embutida dentro do HTML.

Popularidade do PHP

Vários fatores devem ser considerados ao se adotar uma linguagem de programação como PHP ou Java para o desenvolvimento de um projeto ou mesmo como linguagem oficial de desenvolvimento de uma empresa. Exemplos de fatores importantes são a facilidade de aprendizado da linguagem, produtividade da equipe utilizando as linguagens avaliadas e a necessidade de se fazer investimentos em termos de infra-estrutura para executar programas criados naquela tecnologia.
